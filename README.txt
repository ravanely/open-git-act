INTRODUCTION A LA MANIPULATION DES IMAGES AVEC JQUERY
=====================================================

open-Git-act est un simple projet d'introduction à la manipulation
des images avec JQuery.
Il se présente ainsi que suit:
- D'un repertoire css qui contient un fichier bootstrap
- D'un repertoire fonts qui contient les fichiers de polices
- D'un repertoire js qui contient les fichiers js
- D'un fichier index.php qui est le point d'entree de l'application
